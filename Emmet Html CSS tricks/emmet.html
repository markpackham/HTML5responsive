Emmet works with both Html and CSS

        /* Emmet is also great for CSS */
        .class {
          /* pos */
          position: relative;
          /* bgc */
          background-color: #fff;
          /* w10em */
          width: 10em;
          /* h10% */
          height: 10%;
        }
